# 🖥️ SysPanel — Web-Based System Administration Dashboard

A full-stack web dashboard for real-time system monitoring and administration, built with Python Flask.

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Flask](https://img.shields.io/badge/Flask-Web%20Framework-lightgrey)
![SQLite](https://img.shields.io/badge/Database-SQLite-green)
![Platform](https://img.shields.io/badge/Platform-Linux%20%7C%20Windows-orange)

---

## 📌 Features

- 📊 **Live System Metrics** — Real-time CPU, RAM, Disk, and Network monitoring
- 👥 **User Management** — Add, remove, and manage system users
- ⚙️ **Service Management** — Start, stop, and monitor system services
- 🖥️ **Interactive Terminal** — Execute commands directly from the browser
- 📋 **Logs & Monitoring** — View system logs and track activity
- 🔐 **Authentication** — Secure login system

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | Python, Flask |
| Database | SQLite |
| System Metrics | psutil |
| Frontend | HTML, CSS, JavaScript |

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- pip

### Installation

```bash
git clone https://github.com/Moneefakyurek/SysPanel.git
cd SysPanel
pip install -r requirements.txt
python app.py
```

Open your browser: `http://localhost:5000`

Default credentials:
- Username: `munif`
- Password: `admin`
